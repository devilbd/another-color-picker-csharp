# Another Color Picker

A sleek, cross-platform color picker built with .NET 10.0 and Avalonia UI. Designed with a modern, dark theme aesthetic, it provides accurate color picking across various platforms, including native support for Linux (X11/Wayland), Windows, and macOS.

## Features

- **Cross-Platform**: Works seamlessly on Linux (X11 & Wayland), Windows, and macOS.
- **Accurate Color Conversion**: High precision conversions between RGB, HSV, HSL, CMYK, and HEX.
- **Dynamic Previews**: Live visual previews during color selection (e.g., magnifying glass preview).
- **Modern UI**: Built with Avalonia UI and Fluent design concepts, featuring micro-animations, clear states, and custom window styling.
- **Native OS Integration**: Proper native integrations and fallbacks for screen capturing across different display servers.

### Build & Run Locally

To run the application from source during development:

```bash
dotnet run --project AnotherColorPicker
```
